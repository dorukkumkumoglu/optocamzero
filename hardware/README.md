# Optocam Zero Hardware

Everything you need to build an Optocam Zero, the 3D print files, required parts list, and detailed step by step build guide can be found in this hardware folder.

If you're considering building one, check the BOM to get an idea of the cost for required tools and parts. Also, have a look at the build guide, it will give you a clear idea of what the build involves.

## 3D Printing the Parts

All parts are recommended to be printed with PETG for heat resistance. PLA parts may deform under heat generated by the electronics.

The parts are tested with PETG, transparent PETG and PETG-CF.

If you're using a Bambu Lab printer, ready to print Bambu Studio files are provided for Transparent PETG and PETG/PETG-CF. If you're printing transparent PETG, make sure the filament is well dried beforehand for maximum transparency.

All front and back pieces print flat on the plate, so whatever texture you have on your plate will be carried over to the front and back surfaces of your camera. For transparent PETG printing, I suggest using some kind of smooth plate if you want to achieve maximum transparency. Default textured plates will create a frosted look on transparent PETG, which has its own charm.

Two color 3D printing is only needed for the front cover logo and the two colored front spacer. If you don't want to have these, you can print all the pieces single color.

The optional case is printed with TPU. If you haven't printed with TPU before, check general TPU printing instructions.

<br>

## Before you start

Make sure you have all the required parts, tools and 3D printed components ready before you begin.

Read through all the steps in the build guide before starting. Some steps have a specific order that can't be reversed easily.

This build requires basic soldering and desoldering. If you haven't soldered before, it's worth practicing on some scrap components first.

Take your time, especially with the ribbon cable (order extras to be safe) and the header pin soldering/desoldering. These are the most delicate parts of the build and mistakes here are difficult to undo. Be patient with it.
